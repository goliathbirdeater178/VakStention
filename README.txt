How to install this extension on Google Chrome:
1) Download these files as a zip and extract to a specific folder
2)	Type 'chrome://extensions' into the Chrome address bar
3)	At the very top of the extensions menu, check the box for "Developer Mode" if you haven't already done so
4) Click "Load unpacked extension" and select the extracted zip folder
5) Click "Enable" on the extension labeled "VakStention"
